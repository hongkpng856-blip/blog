# 部落格每週更新計劃

## 更新頻率
- **每週至少 1 篇新文章**
- **每週檢查並回覆留言**

## 文章主題輪替

| 週次 | 主題類型 | 範例標題 |
|------|----------|----------|
| 第1週 | 技術教學 | Git 進階技巧、網頁開發實戰 |
| 第2週 | 投資理財 | 被動收入、投資工具比較 |
| 第3週 | 生活分享 | 效率工具、生活小撇步 |
| 第4週 | 健康學習 | 健康習慣、學習方法 |

## 文章品質標準

### 基本要求
- [ ] 文章字數：800-1500 字
- [ ] 至少 1 張特色圖片（800x400px）
- [ ] 正確設定分類和標籤
- [ ] 檢查錯字和語句通順

### 進階優化
- [ ] 加入內部連結（連結到其他文章）
- [ ] 加入相關外部參考連結
- [ ] 加入程式碼範例（如適用）
- [ ] 加入圖表或截圖說明

## 每週維護檢查清單

### 週一：內容規劃
- [ ] 決定本週文章主題
- [ ] 收集素材和參考資料
- [ ] 擬定大綱

### 週三：文章撰寫
- [ ] 撰寫文章草稿
- [ ] 製作或選擇特色圖片
- [ ] 設定分類和標籤

### 週五：發布與推廣
- [ ] 最終校對
- [ ] 發布文章
- [ ] 分享到社交媒體

### 週末：數據檢視
- [ ] 查看 Google Analytics（如有）
- [ ] 檢視熱門文章表現
- [ ] 回覆讀者留言

## 備份計劃

### 每週備份
```bash
# 備份整個部落格目錄
cp -r /home/claw/.openclaw/workspace/blog-repo /home/claw/.openclaw/workspace/backup/blog-repo-$(date +%Y%m%d)
```

### 每月歸檔
- 將備份壓縮並移至長期儲存
- 清理超過 3 個月的舊備份
