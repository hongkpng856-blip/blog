# PythonAnywhere 部署指南

## 步驟

### 1. 註冊 PythonAnywhere 帳號
- 前往：https://www.pythonanywhere.com/
- 選擇「Beginner」方案（免費）
- 註冊帳號

### 2. 上傳網站程式碼
- 登入後，點擊「Files」
- 上傳 `mock_website.py` 或建立新檔案

### 3. 建立 Web App
- 點擊「Web」tab
- 選擇「Add a new web app」
- 選擇「Flask」框架
- Python 版本：3.10

### 4. 設定網站
- 將 `mock_website.py` 內容複製到 `flask_app.py`
- 設定 static files 路徑

### 5. 啟動網站
- 點擊「Reload」按鈕
- 網站 URL：`https://your-username.pythonanywhere.com`

## 預計時間：~15 分鐘
